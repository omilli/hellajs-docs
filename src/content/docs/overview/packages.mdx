---
title: Packages
description: HellaJS is modular, additional packages extend core reactivity.
---

HellaJS is modular and can be used in various ways. The core package provides reactive functionality like signals and effects. Additional packages extend its capabilities.

:::caution[Dependencies]
`@hellajs/core` is a required dependency for other HellaJS packages.
:::
