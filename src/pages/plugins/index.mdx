## TypeScript Configuration

All plugins support TypeScript out of the box. Add HellaJS types to your project:

```json
// tsconfig.json
{
  "compilerOptions": {
    "jsx": "preserve",
    "types": ["@hellajs/dom"]
  }
}
```
